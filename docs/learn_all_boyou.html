load_data()[source]¶

The load_data function loads data according to a list of files specified in the list. Benignware data are vertically stacked, and it is non-discriminative towards different binaries. It initiates the report of current experiment.

Parameters:
  • self.report (dict) – The report is the dictionary of all the results.
  • self.benign_dict (dict) – The benignware names are recorded. The value of each entry is boolean. The purpose of this is to keep track of benignware.
  • self.malware_dict (dict) – Similar to benignware, this works for malware.
  • self.benign_data (array) – Every feature vector in the numpy array is vertically stacked.
  • self.malware_data (array) – Similar to self.benign_data
  • self.benign_label (list) – Labels for benignware [1, 1, 1, … 1]
  • self.malware_label (list) – Similar to benign_label [0, 0, 0, … 0]
  • self.data (array) – Stacking benign data over malware data
  • self.label (array) – Stacking benign label over malware label
